📝 Basic (Ctrl+Shift+...):
• U - UPPERCASE | L - lowercase | T - Title Case | S - Sentence case

💻 Programming (Ctrl+Alt+...):
• C - camelCase | P - PascalCase | N - snake_case

🚀 Special:
• Ctrl+Shift+F - Cycle formats | Ctrl+Shift+G - Smart suggestions
